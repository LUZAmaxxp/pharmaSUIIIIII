Répartition organisationnelle du projet
Le projet sera réalisé par 3 sous-équipes de 5 étudiants :

Équipe A : Gestion Patient
Équipe B : Gestion Pharmacien
Équipe C : Administration, Sécurité et Qualité
Chaque équipe est responsable de :

développement backend
tests unitaires
documentation API
intégration Git
Les composants transversaux (JWT, modèles partagés, sécurité, qualité) seront mutualisés.
